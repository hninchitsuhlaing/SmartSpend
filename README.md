# 💰SmartSpend Personal Finance & Budget Management System

A modern full-stack personal finance management application designed to help users track income, expenses, budgets, and financial insights through an intuitive dashboard.

## My Role

Frontend Developer

Developed responsive user interfaces, dashboard pages, forms, and financial visualization components while collaborating on backend integration.

🚀 Features

- User Authentication
- Expense Tracking
- Income Management
- Budget Planning
- Financial Dashboard
- User Settings

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- Flask
- MySQL
- JWT Authentication
